You can write your "styled components" like this:

```jsx
import { styleql } from "@styleql/react"

const StyleQLButton = styleql.button`
  query ButtonStyles($color: TextColors) {
    typography {
     textColor(color: $color, focus: "indigo-200", hover: "purple-900")
     fontFamily(font: MONO)
    }
  }
`
```

At build time this gets transformed into:

```jsx
const Button = (
  { children, className, styleql: { color }, ...props } = {
    styleql: {
      color: "GREY_900",
    },
  }
) =>
  React.createElement(
    "button",
    Object.assign({}, props, {
      className: `text-${color} hover:text-purple-900 focus:text-indigo-200 font-mono ${className}`,
    }),
    children
  )
```
